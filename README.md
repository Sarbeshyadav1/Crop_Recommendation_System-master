## 📖 Overview

The **Crop Recommendation System** predicts the most suitable crop to cultivate based on key agricultural parameters:

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH value
* Rainfall

By leveraging machine learning algorithms, this system provides accurate crop suggestions, helping optimize agricultural practices.

## 📂 Dataset

The dataset contains agricultural parameters such as:

| ----------- | -------------------------- |

| Humidity    | Relative humidity (%)      |

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Crop_Recommendation_System.git
```

2. Navigate to the project directory:

```bash
cd Crop_Recommendation_System
```

3. Install required dependencies:

```bash
pip install -r requirements.txt
```
